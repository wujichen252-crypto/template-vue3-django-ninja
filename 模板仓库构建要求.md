### 三、后端具体要求（Django-Ninja）

**技术栈**：Python 3.11 + Django 5.0 + Django-Ninja 1.1 + PostgreSQL + Redis + PyJWT + Pydantic + Whitenoise + Gunicorn

#### 3.1 配置结构（`config/settings/`）

**`base.py` 必须包含**：
- `INSTALLED_APPS`：包含 `apps.users`, `ninja`
- `DATABASES`：PostgreSQL 配置，从环境变量读取
- `CACHES`：Redis 配置，默认缓存后端
- `NINJA`：Django-Ninja 全局配置（如 `docs_url='/docs'`）
- `AUTH_USER_MODEL = 'users.User'`（自定义用户模型）
- `LOGGING`：JSON 格式日志，统一 `request_id` 注入
- `MIDDLEWARE`：自定义 `RequestIDMiddleware`（生成/透传 request_id）

**`local.py`**：`DEBUG = True`，允许 `localhost:5173` CORS，启用 Django Debug Toolbar

**`production.py`**：`DEBUG = False`，`ALLOWED_HOSTS` 从环境变量读取，Whitenoise 托管静态文件

#### 3.2 用户模块（`apps/users/`）

**`models.py`**：
```python
class User(AbstractBaseUser):
    username = models.CharField(max_length=32, unique=True, verbose_name="用户名")
    email = models.EmailField(max_length=128, unique=True, verbose_name="邮箱")
    avatar_url = models.URLField(max_length=500, blank=True, null=True, verbose_name="头像")
    status = models.PositiveSmallIntegerField(default=1, verbose_name="状态")  # 1正常 0禁用
    created_at = models.DateTimeField(auto_now_add=True, verbose_name="创建时间")
    updated_at = models.DateTimeField(auto_now=True, verbose_name="更新时间")
    # 软删除：Django 默认无 deleted_at，如需使用 django-parler 或自实现
```

**`schemas.py`**（Ninja 的 Pydantic 模型）：
| Schema | 用途 | 字段 |
|--------|------|------|
| `RegisterIn` | 注册请求 | `username: str`, `password: str`, `email: str` |
| `LoginIn` | 登录请求 | `username: str`, `password: str` |
| `UserOut` | 用户响应 | `id: int`, `username: str`, `email: str`, `avatar_url: str`, `status: int`, `created_at: datetime` |
| `TokenOut` | Token响应 | `access_token: str`, `refresh_token: str`, `expires_in: int` |

**`api.py`**（Ninja Router）：
```python
from ninja import Router
router = Router()

@router.post("/auth/register", response=StandardResponse[UserOut])
def register(request, payload: RegisterIn): ...

@router.post("/auth/login", response=StandardResponse[TokenOut])
def login(request, payload: LoginIn): ...

@router.get("/users/me", response=StandardResponse[UserOut], auth=JWTAuth())
def get_current_user(request): ...

@router.post("/auth/refresh", response=StandardResponse[TokenOut])
def refresh_token(request, refresh: str): ...
```

**`services.py`**（业务逻辑层）：
- `UserService.register(data)`：查重 → bcrypt 哈希密码 → 创建用户
- `UserService.authenticate(username, password)`：查用户 → 校验密码 → 生成双 Token
- `UserService.get_user_by_id(user_id)`：缓存优先查 Redis，回源数据库

#### 3.3 核心基础设施（`core/`）

**`auth.py`**：
- `JWTAuth(HttpBearer)`：Ninja 认证类，从 `Authorization: Bearer <token>` 解析
- `generate_access_token(user_id)`：有效期 2 小时
- `generate_refresh_token(user_id)`：有效期 7 天
- `decode_token(token)`：验证签名、过期时间，返回 payload

**`response.py`**（统一响应）：
```python
from ninja import Schema
from typing import Generic, TypeVar

T = TypeVar('T')

class StandardResponse(Schema, Generic[T]):
    code: int = 200
    data: T = None
    msg: str = "ok"
    request_id: str = ""
```

**`exceptions.py`**（全局异常）：
- 使用 Ninja 的 `@api.exception_handler` 捕获：
  - `ValidationError` → 400，返回具体字段错误
  - `AuthenticationError` → 401
  - `PermissionError` → 403
  - `ObjectDoesNotExist` → 404
  - 其他异常 → 500，隐藏细节，只返回 `msg: "服务器内部错误"`

**`pagination.py`**：
- 封装 `LimitOffsetPagination`，默认 `limit=20`, `offset=0`
- 响应格式：`{ "code": 200, "data": { "items": [], "total": 100 }, "msg": "ok" }`

#### 3.4 路由注册（`config/urls.py`）

```python
from ninja import NinjaAPI
from apps.users.api import router as user_router

api = NinjaAPI(
    title="Template API",
    version="1.0.0",
    description="Django-Ninja 全栈模板接口",
    docs_url="/docs",        # Swagger UI 地址
    openapi_url="/openapi.json"
)

# 统一响应包装（可选中间件或装饰器方式）
api.add_router("/api/v1", user_router)

urlpatterns = [
    path("admin/", admin.site.urls),
    path("", api.urls),       # 所有 ninja 路由挂载到根，前缀已在 router 中定义
]
```
